# Build

Install `docker` and `docker-compose`

Run command:

`bash install.sh`

---

`docker-compose up --build -d`
`docker exec -ti solana bash`
`sh install.sh`
`cd /root/build/solana-release/ && export PATH=$PWD/bin:$PATH && cd .. && cd .. && solana --version`
